## Descripcion

LearnBase es un sistema de gestion de aprendizaje (LMS) dise√±ado para que organizaciones, creadores de contenido y empresas puedan lanzar su propia academia online de forma independiente.

Cada cliente opera en su propio subdominio con usuarios, cursos y contenido completamente aislados. Esto permite a multiples organizaciones utilizar la misma infraestructura mientras mantienen una experiencia personalizada para sus estudiantes.

## Caracteristicas principales

- **Multi-tenancy**: Cada organizacion tiene su propio espacio aislado con subdominio personalizado
- **Gestion de cursos**: Creacion y organizacion de cursos con modulos y lecciones
- **Contenido multimedia**: Soporte para videos con almacenamiento en la nube
- **Analisis de video con IA**: Generacion automatica de titulos y descripciones a partir del contenido de los videos
- **Roles y permisos**: Sistema de roles para administradores, instructores y estudiantes
- **Internacionalizacion**: Soporte para multiples idiomas
