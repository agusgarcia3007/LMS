## Content and Script Generators

These tools help you write course content: scripts, descriptions, materials, summaries.

The most popular. Free plan with GPT-4o mini. Great for scripts, summaries, structuring modules. No message limit but shorter responses than paid.

## Bonus: ChatGPT Prompts for Course Creation

Copy and paste these prompts to speed up your content creation.

**Structure a course:**
"Act as an expert instructional designer. Create the structure for a course about [TOPIC] for [AUDIENCE]. Include modules, lessons per module, learning objectives and estimated duration for each lesson."

**Generate lesson script:**
"Write a script for a 10-minute video lesson about [TOPIC]. Include: opening hook, main points with examples, practical activity and summary. Tone: conversational but professional."

**Create quiz:**
"Generate a 10-question quiz about [LESSON TOPIC]. Include: 5 multiple choice, 3 true/false, 2 short answer. Provide correct answers and explain why."

**Write course description:**
"Write a sales description for a course about [TOPIC]. Include: problem it solves, main benefits, who it's for, what they'll learn, and a call to action. Maximum 300 words."

**Generate lead magnet ideas:**
"Give me 10 lead magnet ideas to attract students to a course about [TOPIC]. For each idea include: title, format (PDF, video, etc.), main content and how it connects to the course."

**Create launch plan:**
"Create a 2-week launch plan for a course about [TOPIC]. Include: social media content, emails, key dates, and tactics to generate urgency."

**Improve explanations:**
"I have this explanation of [CONCEPT]: [YOUR EXPLANATION]. Improve it to be clearer and more memorable. Use analogies and everyday examples."

**Generate practical assignments:**
