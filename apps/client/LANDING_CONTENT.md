- Subtle animated effect

- Light muted background (bg-muted/30)
- Simple bordered cards for problems
- Centered solution with badge

### Features
- Simple grid layout (2x3 on desktop)
- Icons with primary/10 background
- No decorative grid patterns

- Card with ring styling
- Dashed borders between items
- Accordion with single collapsible

- Primary: Blue tones
